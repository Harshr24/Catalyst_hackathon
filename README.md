## Overview
EduMentor AI is an interactive educational assistant providing an AI-powered learning companion that helps students with concept explanations, study notes generation, quizzes, and interactive educational games â€” all through a clean and user-friendly interface.
The system leverages **large language models (LLMs)** integrated via **LangChain**, deployed using **Streamlit**, and designed to support both online and offline educational use cases.

---

## Problem Statement
Students often struggle to find personalized, interactive, and accessible learning support that adapts to different study needs such as revision, self-assessment, and conceptual clarity.

EduMentor AI addresses this by offering:
- On-demand explanations
- Automated study notes
- Quiz generation and evaluation
- Interactive learning activities

---

## Key Features
- **Conversational AI Tutor** for answering academic questions  
- **Automated Study Notes** generation with download support  
- **Quiz Generation & Evaluation** for self-assessment  
- **Interactive Educational Games** for engaging learning  
- **Context-Aware Conversations** using memory buffers  
- **Voice Input Support** for hands-free interaction  
- **Session-Based Chat History** using Streamlit session state  

---

## Tech Stack
- **Programming Language:** Python  
- **Frontend:** Streamlit  
- **LLM Orchestration:** LangChain  
- **LLM Providers:**  
  - Groq (LLaMA-3.3-70B SpecDec)  
  - Ollama (local/offline support)  
- **AI Memory:** ConversationBufferMemory  
- **Dev Environment:** GitHub Codespaces / Dev Containers
